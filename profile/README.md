# Pixium

Open-source collaborative pixel art canvas built on Stellar.

Pixium is a multiplayer on-chain art game where players place pixels on a shared canvas, compete on leaderboards, complete quests, and earn rewards through Soroban smart contracts on the Stellar network.

The project combines blockchain, gaming, and community creativity into a shared experience where every pixel contributes to a constantly evolving canvas.

### Core Features

* Shared collaborative pixel canvas
* Quests and progression system
* Factions and team-based competition
* Global leaderboards
* NFT canvas snapshots and collectibles
* Community-driven color voting

### Project Structure

* `onchain` — Soroban smart contracts written in Rust
* `backend` — Node.js + TypeScript API and WebSocket services
* `frontend` — Next.js application for the player experience
* `indexer` — Stellar event indexer and sync services

### Tech Stack

* Rust + Soroban SDK
* Node.js + TypeScript + NestJS
* Next.js + React + Tailwind CSS
* PostgreSQL + Redis
* Stellar Blockchain

Pixium is community-driven and open to contributors across smart contracts, backend infrastructure, frontend development, and game design.
